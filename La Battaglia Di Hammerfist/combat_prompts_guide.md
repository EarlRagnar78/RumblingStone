# Guida ai Prompt per Sistema di Combattimento di Massa

## Sequenza Consigliata di Richieste

### PROMPT 1: Fondamenta del Sistema
```
Crea un sistema di combattimento di massa per D&D/Pathfinder con le seguenti specifiche:

1. **Meccaniche di Base**
   - Come funzionano i gruppi di combattenti (statistiche aggregate)
   - Sistema di morale e coesione delle unità
   - Regole per il danno di massa vs singoli combattenti

2. **Gestione degli Incantesimi**
   - Come gli incantesimi di supporto/cura influenzano gruppi interi
   - Meccaniche per incantesimi di danno su larga scala
   - Esempi pratici con incantesimi comuni (Fireball, Cure Mass Wounds, Bless, etc.)

Mantieni tutto conciso ma completo. Questo è il primo di una serie di documenti collegati.
```

### PROMPT 2: Statistiche e Nemici Elite
```
Basandoti sul sistema creato nel documento precedente, sviluppa:

1. **Statistiche dei Gruppi**
   - Template per unità base (soldati, arcieri, cavalieri)
   - Modificatori per esperienza e equipaggiamento
   - Come calcolare HP, AC e danni aggregati

2. **Nemici Elite e Capi**
   - Regole speciali per quando i PG prendono di mira leader nemici
   - Azioni speciali disponibili contro nemici elite
   - Sistema di conseguenze tattiche per eliminazione di capi

Fornisci esempi numerici concreti e tabelle di riferimento rapido.
```

### PROMPT 3: Mappe e Posizionamento Tattico
```
Crea il sistema di mappe per il combattimento di massa:

1. **Specifiche Tecniche**
   - Scale in metri per movimenti e distanze
   - Altezze di muri, bastioni e fortificazioni
   - Punti ferita di strutture e ostacoli

2. **Elementi del Campo di Battaglia**
   - Tipi di terreno e loro effetti tattici
   - Impedimenti al movimento e alla visuale
   - Posizioni strategiche (alture, ponti, porte)

3. **Esempio di Mappa**
   - Una battaglia campione con posizioni iniziali
   - Movimenti tatici suggeriti
   - Evoluzione dinamica del campo durante il combattimento

Includi schemi ASCII o descrizioni dettagliate per la visualizzazione.
```

### PROMPT 4: Campagna Completa
```
Sviluppa una campagna di battaglia completa usando i sistemi precedenti:

1. **Scenario di Guerra**
   - Contesto narrativo e fazioni coinvolte
   - Obiettivi strategici per ogni fazione
   - 3-5 battaglie collegate in sequenza

2. **Progressione della Campagna**
   - Come le vittorie/sconfitte influenzano battaglie successive
   - Ricompense e conseguenze per i PG
   - Evoluzione delle forze in campo

3. **Eventi Speciali**
   - Momenti cruciali dove le azioni dei PG cambiano il corso della guerra
   - Alleati che si possono guadagnare o perdere
   - Colpi di scena tattici e narrativi
```

### PROMPT 5: Strumenti del Master
```
Crea gli strumenti finali per gestire il sistema:

1. **Tabelle di Riferimento**
   - Quick reference per tutte le meccaniche
   - Tabelle casuali per complicazioni in battaglia
   - Generatori di eventi tattici improvvisi

2. **Esempi di Gioco**
   - Trascrizioni di round di combattimento campione
   - Decisioni tipiche del master e loro conseguenze
   - Risoluzione di situazioni complesse

3. **Varianti e Opzioni**
   - Regole per diversi tipi di guerra (assedi, battaglie navali, etc.)
   - Adattamenti per diversi livelli di PG
   - Come scalare il sistema per conflitti più grandi o piccoli
```

## Consigli per l'Uso

### Per Ogni Prompt:
- **Riferisci sempre** al documento precedente: "Basandoti sul sistema del documento precedente..."
- **Richiedi esempi concreti** e numeri specifici
- **Chiedi tabelle di riferimento** per consultazione rapida
- **Specifica il tuo sistema** (D&D 5e, Pathfinder, etc.) se relevante

### Gestione della Lunghezza:
- Ogni prompt dovrebbe generare 1500-3000 parole circa
- Se un documento è troppo lungo, chiedi di dividerlo in sottosezioni
- Puoi sempre richiedere espansioni specifiche: "Espandi la sezione sugli incantesimi di cura"

### Personalizzazione:
- Adatta il livello di complessità al tuo gruppo
- Specifica il numero di PG e il loro livello approssimativo
- Indica se preferisci regole più simulate o più narrative

## Prompt Alternativi Brevi

Se preferisci approcci più rapidi:

### Versione Essenziale:
```
Crea un sistema semplificato di combattimento di massa per [TUO SISTEMA] che copra:
- Statistiche base per gruppi di nemici
- Come gestire incantesimi di area
- Una mappa campione con dettagli metrici
- Regole per nemici elite

Mantieni tutto su 2000 parole max, con focus sulla praticità.
```

### Versione per Singola Battaglia:
```
Progetta una singola battaglia di massa dettagliata con:
- 2-3 tipi di unità per lato
- Mappa con dimensioni precise e ostacoli
- Regole speciali per questa specifica battaglia
- Come i PG possono influenzare l'esito

Focus su un combattimento memorabile piuttosto che un sistema completo.
```
