# Bestiario/tokens — arte e token per mostri e PNG

> Immagini `.webp` (ritratti/token da tavolo virtuale). Lotto **L4** del piano
> libreria. Le schede che hanno un token lo dichiarano col campo `**Token**:`.

## Struttura

| Cartella | Contenuto |
|---|---|
| `Monster_And_Png/` | token assegnati/riconoscibili (ex `00_Red Hand Of Doom/Immagini/Monster_And_Png`), incl. `Dragons/`, `Png/`, `immagini_Mostri_D&D_Campagna/` |
| `da-catalogare/` | token con soggetto identificato ma non ancora legato a una scheda (ex `Png_And_Monster_To_Be_added`) |

(marcatore di fazione Mano Rossa).
